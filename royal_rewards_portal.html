📄 API Documentation

The Royal Rewards API lets SingX automatically post transfer data and credit points to members in real time. Each member has a unique API key.

POST
/api/v1/transfer
Records a completed transfer and credits Royal Rewards points.
// Headers Authorization: Bearer rk_live_YOUR_API_KEY Content-Type: application/json // Request Body { "transfer_id": "TX-2025-001", "user_email": "member@email.com", "amount_sgd": 500.00, "destination_currency": "INR", "singx_reference": "SINGX-REF-20250115", "status": "completed", "timestamp": "2025-01-15T10:30:00Z" }
Response 200
{ "success": true, "points_earned": 75, "total_points": 350, "tier": "Silver", "multiplier": 1.5, "message": "75 Royal Rewards points credited." }
GET
/api/v1/member/:email
{ "email": "member@email.com", "total_points": 350, "tier": "Silver", "multiplier": 1.5, "transfers": 7, "total_sgd": 2400.00 }

🔥 Firebase Real-Time Setup
To upgrade from localStorage to real-time Firebase:
1. Create project at console.firebase.google.com
2. Enable Firestore Database (test mode)
3. Replace the FIREBASE_CONFIG object at the top of this file with your project's config
4. Set USE_FIREBASE = true
5. All user IDs, points and transfers will sync in real time across all devices
